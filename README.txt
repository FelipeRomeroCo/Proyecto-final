Requerimientos del proyecto

Se realizará un tablero que permita ordenar las tareas de SCRUM que tengamos en tres columnas
- To Do (Por hacer)
- In Progress (En progreso)
- Done (Hecho)

Se deberán mover las tareas arrastrando y soltando las tarjetas con el cursor

Cada tarea tendrá la siguiente información:
- Título de la tarea
- Responsable
- Fecha de entrega
- Fecha de creación

Se deben crear tareas nuevas desde un pequeño formulario
